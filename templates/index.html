How to Use

  • Write your Python code in the editor panel
  • Click "Run" to execute your code in an isolated Ubuntu container
  • View the output in the output panel
  • Code execution is limited to 10 seconds
  • Containers are automatically cleaned up after 5 minutes
  • No network access is provided for security reasons
